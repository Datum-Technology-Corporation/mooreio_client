## Directory Structure
* `bin` - Scripts, metadata and other miscellaneous files
* `docs` - Reference documentation
* `examples` - Code samples for adding to this test bench
* `src` - Source code root

> * `tb` - Test Bench
> * `tests` - UVM Tests and Test Suites
